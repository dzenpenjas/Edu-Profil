📘 EduProfil

EduProfil adalah aplikasi manajemen profil siswa berbasis web untuk membantu guru mengenal siswa secara menyeluruh. Aplikasi ini memungkinkan pengelolaan kelas, identitas siswa, keunikan, serta catatan perkembangan secara terstruktur dan mudah diakses.
