# Edu-Profil
EduProfil adalah aplikasi manajemen profil siswa berbasis web yang membantu guru mengenal siswa secara menyeluruh. Aplikasi ini memudahkan pengelolaan data kelas, identitas, keunikan, serta catatan perkembangan siswa secara terstruktur dan mudah diakses, sehingga pembinaan dapat dilakukan lebih tepat, personal, dan berkelanjutan.
